## ops运维管理系统 

###功能
* 命令执行
* 日志查看
* 脚本执行
* 文件传送
* 模块安装
* 代码发布
* 服务器资产
* 网络设备资产
* 监控


###系统环境
* centos6.5 x64
* python 2.6.6

###依赖模块
* django (1.6.11)
* salt (2015.5.5)

###说明
* yum install salt-master，安装好salt-master后即有salt模块
